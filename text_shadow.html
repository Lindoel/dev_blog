Propriedade text-shadow em CSS

Podemos definir uma sombra em qualquer tipo de texto com CSS no HTML. 

                  
                        h1 {
                              text-shadow: 1px 2px 10px red;
                        }

TEXTO DE TESTE

a representação do texto ficará assim, onde 1px empurrará a sombra para o lado direito 2px empurrará a sombra para baixo 10px espalhará a sombra gerando um borrão e red corresponde a cor.
