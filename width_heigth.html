Propriedades width e height em CSS

Podemos definir um tamanho em largura e altura para qualquer caixa em CSS 

                  
                        div {
                              width: 50px
                              height: 100px
                        }

a representação da caixa ficará assim, onde width: 50px corresponde ao tamanho da largura da caixa e height: 100px coreesponde a altura da caixa.
